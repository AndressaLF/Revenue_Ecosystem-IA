Open documentation · Personal research project

About this project

RE-IA is a modular Python laboratory that discovers market pain signals, scores opportunities (EPC, audience fit), and assists content production for ethical affiliate workflows in English-speaking (USD) markets.

It is not a spam bot, vote manipulator, or mass poster. It reads public sources to support human-reviewed publishing decisions.

What it does

  • Ingest public trend and discussion data (e.g. Reddit JSON, Google Trends)
  • Extract pain signals and transactional intent
  • Rank opportunities with deterministic scoring
  • Generate draft reviews with AI (optional) + FTC disclosure

What it does not do

  • Post affiliate links directly on Reddit
  • Scrape private or logged-in content
  • Store Reddit user passwords
  • Exceed API rate limits (cached, throttled requests)

Reddit API usage

This application uses the Reddit API under the Reddit Data API Terms for:

  • Reading public subreddit posts and metadata
  • Researching buyer pain and commercial intent (EN, US market)
  • Rate-limited ingestion into a local SQLite database

Operations dashboard (local only)

The Streamlit dashboard (publish queue, research inbox, Medium wizard) does not run on GitHub Pages. It needs your local Python environment, SQLite database, and .env on your machine.

After cloning the full project locally:

python -m venv .venv
.venv\Scripts\activate          # Windows
pip install -r requirements.txt
python -m dashboard

Then open http://localhost:8501 in your browser.

GitHub Pages (this site) = public transparency for Reddit API registration. Streamlit = private operations on your PC.
